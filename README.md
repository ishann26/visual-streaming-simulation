# 🎬 CloudTube – Video Streaming Simulator

CloudTube is a cloud-based video streaming simulator that demonstrates how cloud computing technologies are used in real-world video platforms like YouTube and Netflix.

---

## 🚀 Live Demo
https://video-streaming-simulator.netlify.app/

---

## 📌 Features

- ▶️ Video streaming from cloud storage
- 📤 Upload videos to cloud
- 🎥 Multiple quality options (1080p, 720p, 360p)
- ⚡ CDN-based fast delivery
- 🔁 Simulate server failure
- 🛡️ Backup server routing (High Availability)

---

## ☁️ Cloud Services Used

### 1. Netlify
- Frontend hosting
- Continuous deployment
- Global CDN for fast delivery

### 2. Supabase Storage
- Stores video files (object storage)
- Provides public URLs for streaming
- Similar to AWS S3

---

## 🧠 Cloud Concepts Demonstrated

- Cloud Hosting
- Cloud Storage
- Content Delivery Network (CDN)
- Adaptive Bitrate Streaming
- Fault Tolerance
- High Availability
- Disaster Recovery

---

## ⚙️ How It Works

1. User opens the website
2. Frontend is served via Netlify CDN
3. User selects or uploads a video
4. Video is stored/retrieved from Supabase Storage
5. Video streams in real-time using browser player
6. Quality can be changed dynamically
7. In case of failure, backup route is activated
